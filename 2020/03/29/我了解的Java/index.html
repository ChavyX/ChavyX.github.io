我了解的Java

​ 我写这篇文章主要想发表下自己对于编程语言的一点谬见，留下一些自己当下在编程方面对编程语言的一点感悟。

​ 说到编程语言，大家最为熟悉的莫过于C ， C++， Java， Python 这些，特别是 Python 在这几年一度翻身成为了最流行的编程语言之一，占据编程语言的排行榜前三的位置。为什么编程语言会有那么多种呢？它们是怎么出现的呢？它们有什么差别？又各有什么优势呢？

​ 编程语言其实很早就已经出现在人们的视野之中了，1946年，世界上第一台通用电子计算机“ENIAC”在美国宾夕法尼亚大学诞生。随后直到1957年，IBM 开发出世界上的第一种高级编程语言 Fortran，从此进入高级语言编程的时代。Fortran 的出现改变了传统的人与计算机的交互方式，将人们从繁琐的劳动中解放出来，将大部分精力投入高级的思考中去，因此Fortran的出现具有划时代的意义。之后的许多语言都受到它的影响。

​ Fortran语言的最大特性是接近数学公式的自然描述，在计算机里具有很高的执行效率。易学，语法严谨。它被广泛地应用于并行计算和高性能计算领域，自诞生以来占据了很长一段时间的优势。一直到1980年才被后来居上的Pascal打下编程语言第一的神坛位置。

(图来自腾讯视频：1965~2019年最流行的编程语言变化)

​ Pascal 发行于1970年，基于ALGOL，是一个有影响力的面向对象和面向过程编程语言，它的诞生是为了鼓励使用结构化编程和数据结构进行良好的编程实践。在高级语言发展过程种，Pascal是一个重要的里程碑。它具有丰富的数据结构和构造数据结构的方法，简明灵活的控制结构，以及编译运行效率高，严格的结构化形式等特点，使得Pascal语言可以被方便地用于描述各种算法与数据结构。而且Pascal其严谨的语法，奠定了一个良好的编程习惯标准，使得后来的编程学习者有了一个参考的先例。

​ 到了1985年左右，Pascal也保不住自己的神坛位置，被我们如今最广为人知的C语言抢占了第一的头衔。

​ C语言，一种通用的面向过程的编程语言，于1969年至1973年间诞生于人间。C语言具有高效、灵活、功能丰富、表达力强和较高的可移植性等特点，备受程序员们青睐。其编译简易，能处理低端存储器但仍保持良好跨平台的特性，以及不需要任何运行环境支持便能运行的特点，一度让C语言成为仅25年来使用最为广泛的编程语言，其影响力持续至今且仍未退却。

​ 如今，我们的很多程序，以及操作系统都是由C语言来实现的。很多学校的第一门编程课就是C语言，作为当代编程的启蒙语言，它主要的优点就在于其入门简单，运行效率高，灵活性强。从我学习C语言到现在，我印象最深刻的就是C语言里面的一个特色，指针，甚有没学指针相当于没学过C语言的言论。而且在C语言之后，很多高级语言都是以C语言为基础的，在我们学习了C语言之后，再学习其他语言能更容易地上手，这也是C语言意义重大的一个方面。

(C语言独自一人战略了半壁江山)

​ 虽然但是，C语言在2001年左右，还是被拉下了神坛。在2016年8月份的TIOBE榜单的数据统计里，C语言市场占比甚至下降至11.303%。

​ C语言下降的一个主要原因是，C语言不适合蓬勃发展的web和移动应用开发领域。此外，C语言不像其他大多数语言(如Java、C++和C#)，虽然其有一个“新”C11标准可用，但只做了微小的变化，C语言应该保持小而快的特点。此外，在其中添加一些C++的特性也没必要，因为C++早就有了，所以C语言的发展有些停滞。C语言发展遇瓶颈的另一个原因是没有大公司推动，Oracle支持Java，微软支持C++，C#和TypeScript，谷歌支持Java、Python、Go，Dart和JavaScript,苹果推动Swift和Objective-C等，但是没有一家大公司公开支持C。这些导致C语言的市场占比下滑剧烈。

​ 好啦，终于轮到Java了！想必说到Java，很多人印象最深刻的肯定就是……

没错就是这杯卡布奇诺！没错就是它！（什么PHP，Java天下第一！）

​ 从我接触Java到现在，也才过了一年半。Java不是当前最好，也不是最火的编程语言，但是它占据着整个市场不小的份额，主要归功于它很多优秀的特点和功能：

  • 面向对象设计
  • 简单性
  • 可移植性与平台无关的特性
  • 高性能
  • 多线程
  • 解释执行
  • 健壮性
  • 安全性
  • 分布式

​ Java是我接触的第一门面向对象的编程语言，也是我目前最喜欢的一门语言（因为我也不会别的hhhhh）。在Java编程学习的过程中，其本身JVM所带来的众多独特优势让编程变得更加简单容易，其强大的社区支持以及众多的框架，例如最经典的Spring系列，这些得天独厚的优势都使得Java开发者的编程变得十分便利。

​ Java也是从2001年开始至2018年一直占领着最流行的程序语言榜首。但是Java的一些特点也不是它独有的，为什么偏偏就它能独占鳌头呢？主要原因还是在于过去的十几年里，互联网快速发展，电商发展迅速，互联网成为了一匹黑马，越来越多的人开始把经济发展的目光转向互联网。凭借这样的时代背景和经济趋势，web和移动应用开发成了最引人注目的领域，也正因如此，Java靠它JavaWeb开发以及分布式支持在这股浪潮里站稳了脚跟，且取得了很好的发展。

但终究时代还是在变迁，互联网即将不再是时代主流，人工智能领域，大数据，机器学习成为了未来的趋势，而事实也在证明，我们在慢慢朝着AI领域一步步靠近，且陆续取得了不错的成绩，开辟了很多先例。Python这门语言，也在AI浪潮的助推下，异军突起，在短短五年内，就挤下了C语言和PHP，并且在2018年打败了C语言和PHP都无法撼动的Java，成为最多人使用的编程语言。

（Python打败了Java，成为最多人使用的编程语言）

Python作为一门新兴的编程语言，机器学习、数据分析、数据处理、Web 开发、企业软件开发，甚至是拼接黑洞照片，Python 的影子无处不在，且在人工智能领域，Python已经远远超过了Java。 在著名的编程语言排行榜网站 TIOBE 上，Python 位居最流行编程语言第三位，仅次于 Java 和 C 语言。

（来源于HELLOGITHUB编程语言月报）

​ 且在2019年，Python 的流行度翻了一番（从 5% 到 10%）。有人预测Python的崛起将在2020年延续，并且缩短与Java和C语言之间的差距。为什么 Python 的势头会如此强劲？因为它的入手门槛低，有一个优秀的社区在支持，并受到数据科学家和新生代开发者的喜爱。

​ 但是对于Java来说，其在企业方面的应用已经十分广泛，微服务，分布式等近年十分火热，且也将会是未来趋势的领域，Java也是占据一个主导的位置。从上面的截图也可以看出，Java霸主的地位仍在，并将在 2020 年继续保持这种地位。

​ JVM 是 Java 的基石，其他编程语言（如 Kotlin、Scala、Clojure、Groovy）也将 JVM 作为运行时。最近，Oracle 修改了 JVM 的许可协议。新的许可协议意味着使用 Java、Kotlin、Scala 或其他 JVM 编程语言的公司需要向 Oracle 支付大额费用。所幸的是，OpenJDK 让 JVM 继续免费。另外，还有其他一些公司为 JVM 提供企业支持。

​ 因为体积和速度方面的问题，基于 JVM 的编程语言并不适合用在今天的无服务器环境中。Oracle 正在推动 GraalVM 计划，旨在让 Java 变得更加敏捷和快速，让它更适合用在无服务器环境中。因为除了 Java，没有其他编程语言可以提供企业级的稳定性和可靠性，所以 Java 将在 2020 年继续占主导地位。

​ 此外，Java的地位也因为有者Spring这一巨头框架的存在。在企业开发领域，Spring有者不可替代的地位。而且Spring自出现到现在时间已经17年了，很多人可能认为Spring时间那么久了该淘汰了，但其实恰恰相反，在技术发展如此快速的现代，Spring能活那么久，说明它的实力和沉淀是值得被肯定的，也说明了Spring在未来仍然会是一个非常重要的存在。

（2020年2月份编程语言走势图，来源于HELLOGITHUB编程语言月报）

（2020年2月份编程语言排行榜，来源于HELLOGITHUB编程语言月报）

总结：其实没有绝对的强者和最强的编程语言，时代在进步，技术在发展，我们的目的总是为了更好的明天，学习更先进的技术。学习永不止境，也无边界，不应该说喜欢一种就膈应其他语言。所有语言都有其优点和优势，存在即意义。互相学习，才能更好的进步。
